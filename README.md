# Debangshu Patra - Professional Portfolio

A modern, responsive portfolio website showcasing the work of Debangshu Patra, an author and content creator from Kolkata, India.

## Features

- **Responsive Design**: Fully responsive and mobile-friendly
- **Modern UI**: Clean, professional design with smooth animations
- **Multiple Sections**: Hero, About, Works, Blog, and Contact sections
- **Social Integration**: Links to Facebook, Blog, and other social platforms
- **Contact Form**: Easy way for visitors to get in touch
- **Fast Performance**: Optimized for quick loading

## Sections

1. **Hero Section** - Eye-catching introduction with call-to-action buttons
2. **About Section** - Personal biography and key achievements
3. **Published Works** - Showcase of published books and creative projects
4. **Blog Insights** - Latest articles and content pieces
5. **Contact Section** - Contact form and social media links
6. **Footer** - Navigation and copyright information

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- Google Fonts (Poppins, Playfair Display)
- Font Awesome Icons

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Or deploy to GitHub Pages, Netlify, or any static hosting service

## Deployment

### GitHub Pages
1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select `main` branch as source
4. Your site will be live at `https://yourusername.github.io/debangshu-portfolio`

### Netlify
1. Connect your GitHub repository to Netlify
2. Netlify will automatically build and deploy
3. Custom domain support available

### Other Hosting
Upload the files to any static hosting service (Vercel, Firebase, etc.)

## Customization

- Edit `index.html` to update content
- Modify `css/style.css` to change colors and styling
- Update `js/script.js` for interactive features
- Replace the avatar image URL with your own
